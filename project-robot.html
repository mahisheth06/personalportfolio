About this project

This project was built for ENED 1022 — Engineering Applications at the University of Cincinnati. Our team of three designed and programmed an EV3 LEGO Mindstorms robot that autonomously sorts shredded disaster relief materials.

The robot follows a closed-loop path using two color sensors mounted facing downward, picks up bins of shredded material from a shredder station, uses weight detection to determine material type (organic vs. metal), and transports them to the correct processing station. A mechanical arm with two touch sensors handles the picking and placing of bins. The robot then autonomously returns to the shredder station to repeat the process.

The goal is to automate the hazardous, repetitive task of sorting disaster debris, allowing relief workers to focus resources more effectively without exposing people to dangerous materials or unstable ground.

Tech & hardware

  • LabVIEW — robot programming language
  • 2x color sensors (facing downward for line detection)
  • 2x touch sensors (on mechanical arm for bin detection)
  • Mechanical arm motor for picking up and moving bins
  • EV3 brick — onboard processing and control
  • Must handle terrain variations up to 2cm height
